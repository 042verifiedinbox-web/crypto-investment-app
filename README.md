# Crypto Investment App

## Overview
The Crypto Investment App is designed to assist users in managing and tracking their cryptocurrency investments. The application provides users with real-time data, investment tracking, and portfolio management features.

## Features
- **Real-time Market Data:** Access current prices, market trends, and historical data for various cryptocurrencies.
- **Investment Tracking:** Easily track and manage your cryptocurrency investments.
- **Portfolio Management:** Organize your investments and view performance metrics.
- **User Authentication:** Secure login system for user accounts.
